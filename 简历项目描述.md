# 项目适配说明（适合写在简历中的“语言面试测试岗”项目经历）

## 项目名称
考研/四六级单词备考智能体

## 岗位适配
适用于：Python后端开发 / AI 应用开发 / 语言测试岗 / 智能体应用开发岗

## 简历版项目描述
基于 FastAPI + Vue + Chroma + MySQL 构建了一个面向考研/四六级单词备考场景的智能体应用系统。系统实现了单词背诵、自动出题、错题复盘、生词收藏和复习计划管理等核心功能，并通过 LLM 提示词工程、知识库检索、上下文管理和多智能体协同机制提升学习体验与个性化推荐能力。

## 个人职责
- 负责后端 API 网关与业务接口的设计与实现；
- 设计并实现 LLM 提示词工程与结构化输出逻辑；
- 搭建上下文管理、短期/长期记忆与知识库检索模块；
- 参与前端页面交互与系统整体流程联调；
- 完成项目的本地部署与启动脚本配置。

## 技术栈
- 后端：Python、FastAPI、SQLAlchemy
- 前端：Vue 3、Vite
- 向量库：Chroma
- 数据库：MySQL
- 智能体能力：LLM、RAG、上下文压缩、记忆管理

## 项目亮点
- 实现了“背诵 → 测试 → 复盘 → 复习计划”完整闭环；
- 引入 LLM 提示词工程与结构化 JSON 输出，提升结果稳定性；
- 集成 Chroma 向量检索，支持单词释义与真题考法的知识问答；
- 采用分层架构设计，便于后续扩展多智能体协同与工具调用能力。
